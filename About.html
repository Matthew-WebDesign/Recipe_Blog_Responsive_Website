About Page

Hello, my name is Matthew and this is my recipe blog. It features 3 recipes that are inspired by menu items on Chili's 3 for me menu. An appetizer, entree, and a dessert. I have been going to Chilis a lot recently so it was the first thing on my mind for a food based project. Sharing how to make this food is also good because I want to learn how to cook these meal options as well.
